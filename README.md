# SunTemp Location Weather

A small browser app that checks live temperature and sunlight details for a searched location or the user's current location.

## Run

Double-click `Open SunTemp.cmd`, or open `index.html` in a browser.

If you prefer a local URL, run this folder's local server:

```powershell
node server.mjs
```

The app uses the free Open-Meteo weather and geocoding APIs directly from the browser, so no API key is needed.
